# JG University Landing Page Redesign

A modern and fully responsive university landing page redesign inspired by the official JG University website.

This project was developed as part of the internship assignment using **Next.js**, **Tailwind CSS**, and **Three.js** for immersive 3D animations and modern UI interactions.

---

## 🚀 Tech Stack

- Next.js
- Tailwind CSS
- Three.js
- Framer Motion
- React Icons

---

## ✨ Features

- Modern & Clean UI Design
- Fully Responsive Layout
- Interactive 3D Background Animations
- Smooth Scroll Navigation
- Animated Hero Section
- Creative Hover Effects
- Optimized Performance
- Mobile Friendly Design
- Glassmorphism UI Elements
- Smooth Transitions & Animations

---

## 📌 Sections Included

- Navbar
- Hero Section
- About Section
- Programs / Features Section
- Testimonials
- Call To Action Section
- Footer

---

## 🎨 UI/UX Improvements

The original website was redesigned with a more modern visual style by adding:

- Advanced animations using Framer Motion
- 3D interactive backgrounds using Three.js
- Modern typography and spacing
- Enhanced responsiveness
- Better color hierarchy and user experience
- Improved section layouts and visual engagement
